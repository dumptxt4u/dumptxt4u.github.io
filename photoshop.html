1. Basic Navigation

  • V – Move Tool

  • Ctrl + Space + Click – Zoom In

  • Alt + Space + Click – Zoom Out

  • Ctrl + 0 – Fit to Screen

  • Ctrl + 1 – Actual Pixels (100% Zoom)

  • H – Hand Tool

2. File Management

  • Ctrl + N – New Document

  • Ctrl + O – Open Document

  • Ctrl + W – Close Document

  • Ctrl + S – Save Document

  • Shift + Ctrl + S – Save As

  • Alt + Ctrl + S – Save for Web

3. Layers

  • Ctrl + J – Duplicate Layer

  • Shift + Ctrl + N – New Layer

  • Ctrl + G – Group Layers

  • Ctrl + E – Merge Layers

  • Ctrl + Shift + E – Merge Visible

  • Ctrl + Alt + G – Create Clipping Mask

  • Shift + Ctrl + G – Ungroup Layers

4. Selection Tools

  • M – Rectangular Marquee Tool

  • L – Lasso Tool

  • W – Magic Wand Tool

  • Ctrl + D – Deselect

  • Shift + Ctrl + D – Reselect

  • Ctrl + Shift + I – Inverse Selection

  • Ctrl + A – Select All

5. Brush Tool

  • B – Brush Tool

  • [ or ] – Decrease/Increase Brush Size

  • Shift + [ or ] – Decrease/Increase Brush Hardness

  • Alt + Right-Click + Drag Left/Right – Brush Size

  • Alt + Right-Click + Drag Up/Down – Brush Hardness

  • Shift + B – Cycle through Brush Tools

6. Eraser Tool

  • E – Eraser Tool

  • Shift + E – Cycle through Eraser Tools

  • Alt + Right-Click + Drag Left/Right – Eraser Size

  • Alt + Right-Click + Drag Up/Down – Eraser Hardness

7. Other Tools

  • C – Crop Tool

  • I – Eyedropper Tool

  • J – Spot Healing Brush Tool

  • O – Dodge Tool

  • P – Pen Tool

  • S – Clone Stamp Tool

  • T – Type Tool

  • U – Shape Tool

  • Y – History Brush Tool

  • G – Gradient Tool

  • K – 3D Tools

  • R – Rotate View Tool

8. Transformations

  • Ctrl + T – Free Transform

  • Shift + Ctrl + T – Transform Again

  • Ctrl + Alt + T – Duplicate & Transform

  • Ctrl + Alt + Shift + E – Stamp Visible

  • Shift + Ctrl + Alt + T – Repeat Transform

9. Blending & Opacity

  • Shift + + / - – Cycle through Blend Modes

  • Shift + Number (e.g., Shift + 5) – Set Opacity (50%)

  • Shift + Alt + Number – Set Fill Opacity

10. Color Adjustments

  • Ctrl + U – Hue/Saturation

  • Ctrl + B – Color Balance

  • Shift + Ctrl + B – Auto Color

  • Shift + Ctrl + L – Auto Levels

  • Shift + Ctrl + M – Auto Contrast

  • Ctrl + L – Levels

  • Ctrl + M – Curves

  • Shift + Ctrl + U – Desaturate

11. Masking

  • Ctrl + I – Invert Mask

  • Ctrl + Backspace – Fill with Background Color

  • Alt + Backspace – Fill with Foreground Color

  • Shift + Backspace – Fill with Fill Dialog

  • Shift + Ctrl + Alt + E – Create New Layer from Selection

  • Alt + Click on Mask – View Mask

12. History & Undo

  • Ctrl + Z – Undo/Redo

  • Ctrl + Alt + Z – Step Backward

  • Shift + Ctrl + Z – Step Forward

  • F12 – Revert to Last Save

13. View and Interface

  • Tab – Show/Hide Panels

  • Shift + Tab – Show/Hide All Except Toolbox

  • F – Cycle Screen Modes

  • Shift + F – Full-Screen Mode

  • Ctrl + R – Show/Hide Rulers

  • ; (Semicolon) – Show/Hide Guides

  • Shift + ; – Lock Guides

  • Ctrl + ; – Show/Hide Grids

  • Ctrl + Alt + ; – Clear Guides

  • Shift + Ctrl + ; – Snap to Guides

14. Miscellaneous

  • D – Reset Foreground/Background Colors

  • X – Swap Foreground/Background Colors

  • Q – Quick Mask Mode

  • Ctrl + H – Hide Selection Border

  • Ctrl + Alt + Shift + S – Save for Web
