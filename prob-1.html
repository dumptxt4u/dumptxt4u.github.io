History of Probability

The history of probability dates back to the 17th century. It began with games of chance and insurance problems.

Key milestone: In 1654, Blaise Pascal and Pierre de Fermat corresponded on probability problems related to gambling, laying the foundation for modern probability theory.

Further developments occurred in the 18th and 19th centuries, with contributions from mathematicians like Jacob Bernoulli and Pierre-Simon Laplace.

In the 20th century, Andrey Kolmogorov formalized probability with axioms in 1933.

Famous Persons in Probability

  • Blaise Pascal (1623-1662): Co-founder of probability theory through correspondence on gambling problems.
  • Pierre de Fermat (1607-1665): Collaborated with Pascal on early probability concepts.
  • Thomas Bayes (1702-1761): Known for Bayes' theorem.
  • Andrey Kolmogorov (1903-1987): Axiomatized modern probability theory.
  • Carl Friedrich Gauss (1777-1855): Contributed to probability in statistics and normal distribution.
  • Jacob Bernoulli (1654-1705): Law of large numbers.
  • Pierre-Simon Laplace (1749-1827): Bayesian interpretation and central limit theorem.
  • Augustus De Morgan (1806-1871): Worked on probability rules.

Introduction to Probability

Probability is a branch of mathematics that deals with the occurrence of a random event. It quantifies the likelihood of events happening.

In CBSE Class 12, probability is studied as the measure of uncertainty in random experiments, with values between 0 and 1.

Key terms: Experiment, Outcomes, Sample Space, Events.

Sample Space

The sample space (S) is the set of all possible outcomes of a random experiment.

Example: For tossing a coin, S = {Heads, Tails}.

For rolling a die, S = {1, 2, 3, 4, 5, 6}.

Trial

A trial is a single performance of a random experiment.

Example: Tossing a coin once is a trial.

Simple Event

A simple event (or elementary event) is an event that consists of a single outcome from the sample space.

Example: Getting a '3' on a die roll.

Compound Event

A compound event is an event that consists of two or more outcomes from the sample space.

Example: Getting an even number on a die roll: {2, 4, 6}.

Deterministic Experiment

A deterministic experiment is one where the outcome is predictable and certain under given conditions.

Example: Dropping a ball from a height; it will fall due to gravity.

Random Experiment

A random experiment is one where the outcome cannot be predicted with certainty, though all possible outcomes are known.

Example: Tossing a coin or rolling a die.

Event

An event is a subset of the sample space associated with a random experiment.

Events can be simple or compound.

Random Variable

A random variable is a function that assigns a real number to each outcome in the sample space.

Example: In coin toss, let X = 1 for heads, 0 for tails.

It can be discrete or continuous.

Algebraic Property 1: Union of Two Events

The union of events A and B (A ∪ B) is the event that A or B or both occur.

Algebraic Property 2: Intersection of Two Events

The intersection (A ∩ B) is the event that both A and B occur.

Algebraic Property 3: Complement of an Event

The complement (A') is the event that A does not occur.

Algebraic Property 4: Disjoint Events

Disjoint events have no outcomes in common (A ∩ B = ∅).

Algebraic Property 5: Subset Event

A is a subset of B if every element of A is in B.

Algebraic Property 6: Commutative Property (Union)

A ∪ B = B ∪ A

Algebraic Property 7: Associative Property (Union)

(A ∪ B) ∪ C = A ∪ (B ∪ C)

Algebraic Property 8: Distributive Property

A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

Algebraic Property 9: De Morgan's Law

(A ∪ B)' = A' ∩ B'

Algebraic Property 10: Complement Laws

A ∪ A' = S, A ∩ A' = ∅

Algebraic Property 11: Idempotent Law

A ∪ A = A, A ∩ A = A

Algebraic Property 12: Absorption Law

A ∪ (A ∩ B) = A

Algebraic Property 13: Identity Law

A ∪ ∅ = A, A ∩ S = A

Algebraic Property 14: Difference of Sets

A - B = A ∩ B'

Algebraic Property 15: Symmetric Difference

A Δ B = (A - B) ∪ (B - A)

Conditional Probability

Conditional probability of A given B is P(A|B) = P(A ∩ B) / P(B), where P(B) > 0.

It measures the probability of A occurring given that B has occurred.

Example: In a standard deck of 52 cards, let A be drawing an Ace, B be drawing a Spade. P(A|B) = P(Ace and Spade) / P(Spade) = (1/52) / (13/52) = 1/13.

Another example: Probability it rains given cloudy sky. If P(rain and cloudy)=0.4, P(cloudy)=0.5, then P(rain|cloudy)=0.4/0.5=0.8.

Properties of Conditional Probability

  • 0 ≤ P(A|B) ≤ 1
  • P(S|B) = 1, where S is sample space
  • P(A' | B) = 1 - P(A|B)
  • If A and C are disjoint, P(A ∪ C | B) = P(A|B) + P(C|B)

Multiplication Theorem of Probability

P(A ∩ B) = P(A) * P(B|A) = P(B) * P(A|B)

For independent events, P(A ∩ B) = P(A) * P(B)

Mutually Exclusive Events

Events that cannot occur at the same time. P(A ∩ B) = 0.

P(A ∪ B) = P(A) + P(B)

Mutually Independent Events

Events where the occurrence of one does not affect the other. P(A ∩ B) = P(A) * P(B).

A Priori Probability

A priori (classical) probability is based on logical analysis assuming equally likely outcomes.

P(E) = Number of favorable outcomes / Total outcomes.

Example: Probability of heads in coin toss = 1/2.

Examples of Mutually Exclusive Events

  1. Tossing a coin: Heads and Tails (cannot both happen).
  2. Rolling a die: Getting 1 and getting 2 (in one roll).
  3. Drawing a card: Ace of spades and King of hearts.

Differentiation: They have no common outcomes.

Examples of Mutually Independent Events

  1. Tossing two coins: Outcome of first doesn't affect second.
  2. Rolling a die and tossing a coin.
  3. Drawing two cards with replacement.

Differentiation: Probability of joint occurrence is product of individual probabilities.

Relation Between Mutually Independent and Mutually Exclusive Events

Mutually exclusive events are events that cannot happen simultaneously, so P(A ∩ B) = 0.

Mutually independent events satisfy P(A ∩ B) = P(A) * P(B).

Detailed Explanation: If two events A and B are mutually exclusive and both have positive probabilities (P(A) > 0 and P(B) > 0), they cannot be independent. This is because for independence, P(A ∩ B) should equal P(A)P(B) > 0, but for mutual exclusivity, P(A ∩ B) = 0, leading to a contradiction (0 ≠ positive number).

However, if one event has probability 0 (impossible event) or 1 (certain event), it can be both exclusive and independent in trivial cases, but generally, non-trivial mutually exclusive events are dependent.

In summary, mutual exclusivity implies dependence unless one event is impossible.

Bayes' Theorem

Bayes' Theorem: P(A|B) = [P(B|A) * P(A)] / P(B)

It updates probabilities based on new evidence.

Proof of Bayes' Theorem

From definition: P(A|B) = P(A ∩ B) / P(B)

And P(B|A) = P(A ∩ B) / P(A)

So, P(A ∩ B) = P(B|A) * P(A)

Substitute: P(A|B) = [P(B|A) * P(A)] / P(B)

For partitions, P(B) = Σ P(B|A_i) * P(A_i)
