Testing of Hypothesis

Level of Significance (α) – A key concept in Hypothesis Testing

1. What is Hypothesis Testing?

Hypothesis testing is a statistical method used to make decisions about a population parameter based on sample data. It helps us determine whether there is enough evidence to reject or accept a claim (hypothesis).

2. Basic Terms

Term Definition
Null Hypothesis (H₀) The statement of "no effect" or "no difference" (default assumption).
Alternative Hypothesis (H₁ or Hₐ) The statement we are trying to prove (opposite of H₀).
Level of Significance (α) The probability of rejecting H₀ when it is actually true (Type I error). Common values: 0.05 (5%), 0.01 (1%), 0.10 (10%).
Critical Region / Rejection Region The area in the distribution where we reject H₀ if the test statistic falls in it.
P-value The smallest level of significance at which we can reject H₀.

3. Steps in Testing of Hypothesis

  1. State the Null Hypothesis (H₀) and Alternative Hypothesis (H₁).
  2. Choose the Level of Significance (α).
  3. Select the appropriate test (Z-test, t-test, Chi-square, etc.).
  4. Calculate the test statistic.
  5. Find the critical value or P-value.
  6. Make a decision: Reject H₀ if test statistic is in critical region or P-value ≤ α.
  7. Draw conclusion in the context of the problem.

4. Common Levels of Significance

  • α = 0.05 (5%) → Most commonly used in social sciences and engineering.
  • α = 0.01 (1%) → Used when we need very strong evidence (medical, safety-critical fields).
  • α = 0.10 (10%) → Used in exploratory studies.

5. Type I and Type II Errors

Decision H₀ is True H₀ is False
Reject H₀ Type I Error (Probability = α) Correct Decision (Power = 1 - β)
Fail to Reject H₀ Correct Decision Type II Error (Probability = β)

Note for KTU Exams: Always mention that lower α means stricter test (less chance of Type I error but higher chance of Type II error).
