Section 1: General Functions and Methods

  1. 1. What is an advantage of using functions in programming?
    (a) Code repetition   (b) Abstraction   (c) Increased complexity   (d) Slower debugging
    Correct Answer: (b) Abstraction
    Functions promote abstraction by hiding implementation details and enabling code reuse.
  2. 2. What does the term "function signature" refer to?
    (a) The body of the function   (b) The return type of the function   (c) The combination of the function's name and parameter list   (d) The access specifier of the function
    Correct Answer: (c) The combination of the function's name and parameter list
  3. 3. What is the correct syntax for creating an object in Java?
    (a) ClassName objectName = ClassName();   (b) ObjectName = new ClassName();   (c) ClassName objectName = new ClassName();   (d) New ClassName objectName;
    Correct Answer: (c) ClassName objectName = new ClassName();
  4. 4. Which of the following is an example of a non-member method?
    (a) Void display()   (b) Static void checkSum()   (c) Void sum()   (d) Public void calculate()
    Correct Answer: (b) Static void checkSum()
    Static methods belong to the class, not instances (non-member).
  5. 5. What type of function brings about a change in its arguments and is referred to as an impure function?
    (a) Pure function   (b) Static function   (c) Impure function   (d) Reference function
    Correct Answer: (c) Impure function
  6. 6. In Java, how are objects passed to methods?
    (a) Call by Value   (b) Call by Reference   (c) Call by Name   (d) Call by Assignment
    Correct Answer: (a) Call by Value
    Object references are passed by value in Java.
  7. 7. Which of the following is a valid function prototype?
    (a) Public void calculateSum()   (b) Public calculateSum(int a, int b)   (c) CalculateSum(int a, int b)   (d) Public int calculateSum(int a, int b)
    Correct Answer: (d) Public int calculateSum(int a, int b)
  8. 8. Which of these statements is true for method overloading?
    (a) Methods with the same name and identical parameter lists can be overloaded.   (b) Methods with the same name and different parameter lists are considered overloaded   (c) Overloaded methods must return the same data type   (d) Overloaded methods cannot have more than two parameters
    Correct Answer: (b) Methods with the same name and different parameter lists are considered overloaded
  9. 9. Which method call in the following options refers to calling a member method?
    (a) ObjectName.sum()   (b) CheckSum()   (c) Main()   (d) New ClassName();
    Correct Answer: (a) ObjectName.sum()
  10. 10. What is the general format for calling a function that does not return a value?
    (a) Return_type variableName = function_name(actual_parameters);   (b) Function_name(actual_parameters);   (c) Function_name;   (d) Function_name[parameters];
    Correct Answer: (b) Function_name(actual_parameters);
  11. 11. Find the odd one out
    (a) Method overloading   (b) Polymorphism   (c) Abstraction   (d) Multiple forms of same
    Correct Answer: (c) Abstraction
    The others relate to polymorphism.
  12. 12. Built-in functions are also called:
    (a) Library methods   (b) User-defined methods   (c) Fixed methods   (d) All of the above
    Correct Answer: (a) Library methods
  13. 13. How many return statements can a method have and how many of them should get executed for the program to run successfully?
    (a) 1, 1   (b) Many, 1   (c) 1, Many   (d) Many, Many
    Correct Answer: (b) Many, 1
  14. 14. A method in which the method parameters can get modified is called a ______.
    (a) Pure method   (b) Impure method   (c) Virtual method   (d) None of the above
    Correct Answer: (b) Impure method
  15. 15. A method prototype has which of the following components?
    (a) Name of method   (b) Return type of method   (c) List of formal parameters   (d) All of the above
    Correct Answer: (d) All of the above
  16. 16. A method with no result or value to return has a return type as
    (a) Null   (b) Void   (c) No return type   (d) Default
    Correct Answer: (b) Void
  17. 17. When a reference is passed to a method as a parameter, which of the following statement is true?
    (a) The reference parameter creates a copy of the variable passed to it   (b) Reference parameter refers to the same variable passed to it   (c) The reference parameter value can get changed but passing variable value does not get changed   (d) A reference can be of simple data type
    Correct Answer: (b) Reference parameter refers to the same variable passed to it
  18. 18. An operator used to invoke methods or variables using class name
    (a) New   (b) Dot   (c) Relational   (d) Arithmetic
    Correct Answer: (b) Dot
  19. 19. What would be the prototype of the below method for implementing method overloading? int check(double x)
    (a) Double check(int x, double y)   (b) Double check(double x)   (c) Both (a) and (b)   (d) Neither (a) nor (b)
    Correct Answer: (d) Neither (a) nor (b)
    Overloading requires different parameter types or count.
  20. 20. The parameters that are used in the prototype of the method are called
    (a) Actual parameters   (b) Formal parameters   (c) Virtual parameters   (d) Real parameters
    Correct Answer: (b) Formal parameters
  21. 21. Which of the following java statement will compile successfully for defining a method to receive a list of 10 values?
    (a) Void test-method (int a[10])   (b) Void test-method (int a[ ])   (c) Void test-method (int a)0   (d) Both (b) and (c)
    Correct Answer: (b) Void test-method (int a[ ])
  22. 22. The number of values that a method can return is
    (a) 1   (b) 2   (c) 3   (d) 4
    Correct Answer: (a) 1
  23. 23. A student writes the following code to multiply a variable z by 5. Incorrect statement: z * 5;
    A. z *= 5; B. z = z * 5; C. z * 5; (a) Only A (b) Only B (c) Both A and B (d) All the three
    Correct Answer: (c) Both A and B
  24. 24. A student writes the following code to divide a variable a by 4. Incorrect: a = /4;
    A. a /= 4; B. a = a / 4; C. a = /4; (a) Only A (b) Only B (c) Both A and B (d) All the three
    Correct Answer: (c) Both A and B
  25. 25. Increment n by 1. Incorrect: n + 1;
    A. n++; B. n += 1; C. n = n + 1; (a) Only A (b) Only C (c) All A, B, and C (d) None
    Correct Answer: (c) All A, B, and C
  26. 26. Subtract 7 from p. Incorrect: p = 7-;
    A. p -= 7; B. p = p - 7; C. p = -7; (a) Both A and B (b) Only B (c) Only C (d) None
    Correct Answer: (a) Both A and B

Section 2: Constructors

  1. 1. A special method that initializes values to data members is called ______.
    (a) Constructor   (b) Method   (c) Function   (d) All of the above
    Correct Answer: (a) Constructor
  2. 2. Access specifiers can be used for defining constructors.
    (a) True   (b) False   (c) May be   (d) May not be
    Correct Answer: (a) True
  3. 3. Which of the following is a return type of a constructor?
    (a) Void   (b) Int   (c) Null   (d) None of the above
    Correct Answer: (d) None of the above
  4. 4. If a constructor is declared as private, how many instances can be created?
    (a) 1   (b) 2   (c) Many   (d) 0
    Correct Answer: (d) 0
    (From outside the class; used in singleton patterns with static factory)
  5. 5. If no constructor is defined in the class?
    (a) No error. Default values initialized   (b) Error   (c) Impossible   (d) Program hangs
    Correct Answer: (a) No error. Java provides default constructor
  6. 6. Output of: class Abc { int a; String c; ... new Abc(); print a + " " + c }
    (a) 1 * 1   (b) 0 0   (c) 0   (d) 0 null
    Correct Answer: (d) 0 null
  7. 27. Which constructor is automatically invoked if none is defined?
    (a) Parameterized   (b) Default   (c) Private   (d) Static
    Correct Answer: (b) Default constructor
